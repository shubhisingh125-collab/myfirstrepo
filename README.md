Experiment 8 – GitHub and Version Control

In this experiment, I learned how to create a GitHub account and understand the basics of version control using Git. I created my own repository and added files to it. 

I performed basic Git operations such as clone, add, commit, and push. These commands helped me manage my project files and upload them to GitHub.

I also learned how collaboration works by forking a repository and creating a pull request. This experiment helped me understand how developers work together on projects efficiently.

Overall, this practical improved my understanding of Git and GitHub and their importance in modern software development.
